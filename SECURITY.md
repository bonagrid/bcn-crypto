# Security Policy

## Supported Versions
We are actively fixing vulnerabilities only in the following versions of our project:

| Version | Supported     |
| ------- | ------------- |
| x.x     | -             |

## How to Report a Vulnerability
**Please do not open public issues to report vulnerabilities.**

If you have discovered a security issue, please send a detailed description to the lead author at glebobitotsky@yandex.com.

We will try to respond within 48 hours and coordinate the release of a fix. After the issue is resolved, we will publicly thank you in the changelog (Changelog), if you wish.
